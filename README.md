# ansible-role-secured-server

This role means to fully secure a bare-metal server against multiples cyber-risks. 

## Different functional submodules

- packages installation
- ssh server configuration
- user management
- ntp configuration
- password quality management
- unattended upgrades automatisation
- random entropy generation
- ufw firewall configuration
- intrusion tools (crowdsec, psad, prometheus)
- audition (aide, clamav, logwatch, lynis, rkhunter)
- mail client
